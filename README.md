# Casa Inteligente con Arduino

## Descripción
Proyecto desarrollado en Arduino IDE que controla múltiples LEDs usando:
- Sensor LDR
- Pantalla LCD 16x4
- Botones
- Programación orientada a objetos
- Polimorfismo mediante una clase base Componente

## Funcionalidades
- Modo Normal
- Modo Relajación
- Modo Lectura
- Modo Fiesta
- Visualización del LDR en LCD

## Hardware
- Arduino Mega
- LEDs
- Resistencias
- LDR
- LCD 16x4
- Botones
- Switchs
- Cables macho-macho, macho-hembra y cable para proto
